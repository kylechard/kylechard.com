Active Projects

My research shares a common theme of developing systems to support new computational and data-intensive approaches to accelerate scientific discovery. An updated list of active projects is available on the Globus Labs website. Selected projects include:

funcX is a new federated function-as-a-service (FaaS) platform that enables lightweight functions to be dispatched to execute on remote computing endpoints. funcX enables computing to be fluid, such that functions can be executed wherever makes the most sense (e.g., cheapest, fastest, closest to data).
Parsl is a parallel programming library for Python. Parsl provides an intuitive, pythonic way of composing and parallelizing programs by annotating "apps": Python functions or external applications that run concurrently.
Globus is a research data management platform used by more than 150K users around the world. Globus provides a range of services that make it easy to manage, transfer, share, publish, and discover research data irrespective of where data reside.
DLHub is a hosted model publication and on-demand inferenace service. It provides a web-based interface to publish, share, discover, and execute machine learning models.
ExaWorks is an ECP-funded project that aims to both address exascale workflow challenges and integrate robust workflow technologies into an easy-to-use, scalable, and performant workflows SDK.
Whole Tale is a web-based platform for reproducible research that enables the creation, publication, and execution of tales - executable research objects that capture data, code, and the complete software environment used to produce research findings.
WorkflowsRI is a community-centric effort to design a community infrastructure to democratize workflows research by providing a common basis for comparison, evaluation, and verification of workflows research results.
